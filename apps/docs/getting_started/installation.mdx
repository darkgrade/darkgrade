## Installation

Install Darkgrade Fuse using npm:

```bash
npm install @darkgrade/fuse
```

## Requirements

- **Node.js** 18+ or **Bun** 1.0+
- **TypeScript** 5.3+ (recommended)
- A PTP-compatible camera connected via USB
